Use the sidebar menu to access:
- **POS**: Make sales
- **Transactions**: View sales history
- **Returns**: Create return requests

## Point of Sale (POS)

### Making a Sale

#### Step 1: Add Products to Cart
1. Click **POS** in the sidebar
2. Search for products:
   - Type product name in search box
   - Or type SKU/barcode number
   - Or scroll through product grid
3. Click on a product to add it to cart
4. Product appears in cart on the right

#### Step 2: Adjust Quantities
- Click **+** button to increase quantity
- Click **-** button to decrease quantity
- Click **X** button to remove item completely

#### Step 3: Select Customer (Optional)
1. Click customer dropdown at top of cart
2. Search for existing customer
3. Or click "Add New Customer" to create one:
   - Enter customer name
   - Enter phone number
   - Enter email (optional)
   - Click "Create Customer"

#### Step 4: Apply Discount (Optional)
1. Find discount section in cart
2. Select discount type:
   - **Percentage**: Enter % off (e.g., 10 for 10% off)
   - **Fixed Amount**: Enter dollar amount off
3. Discount applies automatically to total

#### Step 5: Checkout
1. Review cart items and total
2. Click **Checkout** button
3. Select payment method:
   - **Cash**: Customer pays with cash
   - **M-Pesa**: Mobile money payment
   - **Bank**: Card or bank transfer
   - **Debt**: Customer owes money
4. For cash payments:
   - Enter amount customer gave you
   - System shows change to give back
5. Click **Complete Sale**
6. Sale is recorded!

#### Step 6: Print Receipt
1. After sale completes, receipt preview shows
2. Click **Print Receipt** button
3. Receipt prints from browser
4. Give receipt to customer

### Quick Tips for POS

**Fast Checkout**
- Use search box for quick product lookup
- Type SKU if you know it
- Use Tab key to navigate quickly

**Handling Mistakes**
- Remove wrong items with X button
- Adjust quantities before checkout
- Can't edit after sale is complete

**Out of Stock Items**
- Products with no stock are grayed out
- Cannot add to cart
- Contact admin to restock

**Multiple Items**
- Add same product multiple times
- Quantities combine automatically
- Or use + button to increase

## Transactions

### Viewing Your Sales

1. Click **Transactions** in sidebar
2. See all completed sales
3. Most recent sales appear first

### Filtering Transactions

**By Date**
1. Click date range dropdown
2. Choose preset:
   - **Today**: Today's sales only
   - **This Week**: Last 7 days
   - **This Month**: Current month
   - **All Time**: Everything
3. Or click "Custom" for specific dates

**By Payment Method**
1. Click payment method dropdown
2. Select: Cash, M-Pesa, Bank, or Debt
3. List filters automatically

**By Search**
1. Type in search box
2. Search by:
   - Customer name
   - Transaction number
3. Results update as you type

### Viewing Transaction Details

1. Click on any transaction in list
2. See complete information:
   - Transaction number
   - Date and time
   - Customer (if recorded)
   - All items sold
   - Quantities and prices
   - Payment method
   - Total amount
   - Discount applied (if any)

### Reprinting Receipts

1. Open transaction details
2. Click **Reprint Receipt** button
3. Receipt preview appears
4. Click **Print Receipt**
5. Give to customer

## Returns

### Creating a Return Request

#### When to Create Returns
- Customer wants to return product
- Product is defective
- Wrong item sold
- Customer not satisfied

#### How to Create Return

1. Click **Returns** in sidebar
2. Click **Create Return** button
3. Select the original transaction:
   - Search by transaction number
   - Or search by customer name
   - Click on correct transaction
4. Select items to return:
   - Click + to increase return quantity
   - Click - to decrease return quantity
   - Can return partial quantities
5. Enter reason for return:
   - Be specific and detailed
   - Explain customer's concern
   - Minimum 10 characters required
6. Review return summary:
   - Check items and quantities
   - Verify total refund amount
7. Click **Create Return Request**
8. Return is submitted for admin approval

### Checking Return Status

1. Go to **Returns** page
2. See all your return requests
3. Status indicators:
   - **Pending** (Yellow): Waiting for admin approval
   - **Approved** (Green): Accepted, refund processed
   - **Rejected** (Red): Not approved

### Return Details

1. Click on any return
2. View information:
   - Return number
   - Original transaction
   - Items being returned
   - Reason provided
   - Current status
   - Who approved/rejected (if decided)

**Note**: Only admins can approve or reject returns. You can only create and view them.

## Common Tasks

### Starting Your Shift

1. Log in to system
2. Go to POS
3. Verify you can access products
4. Test printer if needed
5. Ready to make sales!

### Processing a Quick Sale

1. POS → Search product → Click to add
2. Adjust quantity if needed
3. Checkout → Select payment → Complete
4. Print receipt
5. Done! (Under 30 seconds)

### Handling Cash Payments

1. Add items to cart
2. Click Checkout
3. Select "Cash"
4. Ask customer for payment
5. Enter amount they gave you
6. System calculates change
7. Give change to customer
8. Complete sale
9. Print receipt

### Handling M-Pesa Payments

1. Add items to cart
2. Tell customer the total
3. Wait for M-Pesa confirmation
4. Click Checkout
5. Select "M-Pesa"
6. Complete sale
7. Print receipt

### Handling Returns

1. Listen to customer concern
2. Get original receipt/transaction number
3. Go to Returns → Create Return
4. Find original transaction
5. Select items to return
6. Enter detailed reason
7. Submit for approval
8. Inform customer admin will review
9. Tell them to check back later

### End of Shift

1. No special closing process needed
2. Simply log out
3. System tracks all your sales automatically

## Troubleshooting

### Cannot Find Product
- Try different search terms
- Check spelling
- Try SKU instead of name
- Ask admin if product exists

### Product Won't Add to Cart
- Check if product is in stock
- Look for "Out of Stock" indicator
- Contact admin to restock
- Offer alternative product

### Printer Not Working
- Check printer is on
- Check paper is loaded
- Try printing from browser
- Contact admin if persists

### Wrong Item Added
- Click X button to remove
- Add correct item
- Complete sale normally

### Customer Wants Refund
- Create return request
- Explain admin must approve
- Give them return number
- Tell them to check back

### Forgot Password
- Click "Forgot Password" on login
- Or contact admin
- Admin can reset your password

### System Running Slow
- Refresh the page
- Close other browser tabs
- Check internet connection
- Contact admin if continues
